# INTRODUZIONE
Lo schema E/R (Entity - Relationship) è una rappresentazione grafica dei concetti della realtà che si analizza. Questo modello permette di rappresentare i concetti (entità) e le associazioni tra esse; inoltre vengono anche rappresentati gli attributi, ovvero delle caratteristiche che appartengono alle entità. Per ogni entità c’è un attributo chiave, ovvero un attributo univoco per ogni istanza, rendendo possibile identificarla.

Ci sono più notazioni per realizzare uno schema E/R, le due più utilizzate sono “Chen” e  “crow’s foot”. La principale differenza tra le due è la rappresentazione delle diverse associazioni tra le entità.

Nella fase di “traduzione” da linguaggio naturale a modello E/R si usano indicativamente queste regole:

| Struttura grammaticale | Struttura E/R |
| sostantivo (di un concetto complesso che va approfondito) | entità |
| nome proprio | istanza |
| aggettivo o sostantivo (di un concetto semplice) | attributo |
| verbo | associazione |
| avverbio | attributo di un’associazione |
